# Taiwanese Text 臺語文字

現今仍通行著三種文字系統：漢字、台羅與白話字（教羅）。

書寫上則有全漢、全羅、漢羅並行三種派別，而漢字兩種羅馬字排列組合下來共有五種書寫方式。
