# Hakka 客語

### Accent 腔調
1. 四縣：桃園、苗栗、屏東、高雄的六堆地區
2. 海陸：桃園、新竹、宜蘭、花蓮、臺東部分地區
3. 大埔：臺中東勢一帶
4. 饒平：新竹竹北、苗栗卓蘭、中壢一帶
5. 詔安：雲林二崙崙背一帶
